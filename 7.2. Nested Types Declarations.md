#### Nested Types
A type declared inside another type is known as a *nested type* 
Generally:
- The name is effectively *OuterName.NestedName*
	- but this can be shortened in the scope of OuterName
- Nested types can generally have any accessibility: public, protected, default and private
- Enclosing and nested classes all share privileged access to members

##### Quick question
How should you instantiate the class Nested from an entirely different scope?
```Java
class Outer {
	static class Nested {}
}
```
Answer: `new Outer.Nested`
(assuming necessary import)
