#### Downstream Collectors
The items selected by a `groupingBy` operation form a sequence of their own

It can be useful to process those items in stream-like ways rather than simply add them to a list in the value side of the map 
This is done using a *downstream collector*

![[Pasted image 20260518140954.png]]

Example code:
```java
package original.src.main.java.original;  
  
import java.util.Collections;  
import java.util.List;  
import java.util.stream.Collectors;  
import java.util.stream.Stream;  
  
class OStudent {  
  private String name;  
  private int percent;  
  private List<String> courses;  
  
  public OStudent(String name, int percent, String... courses) {  
    this.name = name;  
    this.percent = percent;  
    this.courses = List.of(courses);  
  }  
  
  public String getName() {  
    return name;  
  }  
  
  public void setName(String name) {  
    this.name = name;  
  }  
  
  public int getPercent() {  
    return percent;  
  }  
  
  public void setPercent(int percent) {  
    this.percent = percent;  
  }  
  
  public List<String> getCourses() {  
    return courses;  
  }  
  
  public void setCourses(List<String> courses) {  
    this.courses = courses;  
  }  
  
  @Override  
  public String toString() {  
    return "OStudent{" +  
            "name='" + name + '\'' +  
            ", percent=" + percent +  
            ", courses=" + courses +  
            '}';  
  }  
}  
  
public class DownstreamCollecting {  
  public static String letterGrade(OStudent s) {  
    int percent = s.getPercent();  
    if (percent > 90) return "A";  
    if (percent > 80) return "B";  
    if (percent > 70) return "C";  
    return "D";  
  }  
  public static void main(String[] args) {  
    Stream.of(  
            new OStudent("Fred", 87, "Math", "Physics"),  
            new OStudent("Jim", 77, "Art", "Geography"),  
            new OStudent("Sheila", 98, "Math", "Physics", "Quantum Mechanics", "Astronomy"),  
            new OStudent("Alice", 85, "Math", "Computing", "Algorithms"),  
            new OStudent("Mary", 79, "History"),  
            new OStudent("John", 92, "Geography", "Stellar Cartography", "Navigation")  
    )  
            .collect(Collectors.groupingBy(s -> letterGrade(s),  
                    Collectors.mapping(s -> s.getName(), Collectors.toList())))  
            .forEach((k, v) -> System.out.printf("Students with grade %s are %s\n", k, v));  
  }  
}
```

##### Quick Question
Assuming the `Customer` class works as expected, what happens?
```java
System.out.println(Stream.of(
	new Customer("Fred", "North"),
	new Customer("Alice", "North"),
	new Customer("William", "East")
	).collect(Collectors.groupingBy(
		Customer::getRegion,
		Collectors.mapping(Customer::getName,
	Collectors.joining(", ")))));
```

Answer: `{North=Fred, Alice, East=William}`
