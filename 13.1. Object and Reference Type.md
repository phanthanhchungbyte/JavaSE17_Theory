#### Reviewing Assignment Compatibility
A reference of type P can refer to an object if that object "is a" P
- either of class P
- or of a subclass of P
- or of a class that implements P
This means that any given reference could refer to something "more specific"
- but the compiler only directly permits use of features of P
- to access the specialized features of the object, we must cast

#### Form and Effect of Casting
To perform a reference type cast
- prefix an expression with the desired type surrounded by parentheses
```Java
Object obj = "Hello";
String s = (String)obj;
```
The effect of a successful cast is to create a new expression with the "value" of the original, but with a new type
- The value of a reference expression is the reference value, perhaps the memory address, or a proxy for it
- The resulting expression is like any other reference, it can be used for method invocation, field access, and more

#### Testing Object Type with instanceof
The assignment compatibility of an object at runtime can be tested using the `instanceof` operator
`<reference expression> instanceof <classname>`
- The `<classname>` is the *name*  of the class, not an expression of type `java.lang.Class`
- If `<reference expression`> is `null`, the result is `false`
- `<classname>` can describe an interface or enum too
- Tests that cannot possibly be true can be rejected by the compiler
```Java
public class TryCasts {  
  public static void main(String[] args) {  
    CharSequence cs = "";  
    CharSequence[] csa = {cs};  
    String s1 = (String)cs + 3;  
    System.out.println(s1);  
    String s2 = (String)csa[0] + 3;  
    System.out.println(s2);  
    int l1 = ((String)cs).length();  
    System.out.println(l1);  
  }  
}
```

##### Quick Question
Given:
```java
class P {
	void doOne() {}
	void doTwo() {}
}
class C extends P {
	void doTwo() {}
	void doThree() {}
}
```
and:
```java
P p = null;
```

Which compile?
```java
A) p.doOne();
B) p.doTwo();
C) p.doThree();
D) (C)p.doTwo();
E) ((C)p).doTwo();
F) ((C)p).doThree();
```
Answer: A, B, E, F
