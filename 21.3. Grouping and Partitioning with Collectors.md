#### Tidying the Collection Mechanism
The basic collect operation takes three arguments
- these are typically lambda expressions
- they can be "messy"

The `Collector` interface essentially aggregates these three,
and a couple of supporting features into a single interface

The `Collectors` class contains factories for several
implementations built into the APIs

The stream API provides `collect(Collector c)` to use these

#### Key Collectors Factories
The Collectors class provides several Collector factories.
Key among these are
- `groupingBy` -- builds a map with programmer-defined keys
- `partitioningBy` -- builds a map keyed with true and false
- `toList` -- adds the stream items to a list
	- `toUnmodifiableList`
- `toSet` adds the stream items to a set 
- `toMap` -- provides lower level programmer control over populating the map

Example code:
```java
package original.src.main.java.original;  
  
import java.util.List;  
import java.util.stream.Collectors;  
  
public class SimpleCollection {  
  public static void main(String[] args) {  
    List.of("Fred", "Jim", "Sheila", "Bill", "Joe", "Sally", "John", "Tommy")  
            .stream()  
            .collect(Collectors.groupingBy(s -> s.length()))  
            .forEach((k, v) -> {  
              System.out.printf("%d character words are %s\n", k, v);  
            });  
  }  
}
```

##### Quick Question
Given a stream containing `Customer` objects, and an instance
method on `Customer:String getRegion()`, what will be
the type of the variable `result`:
```java
var result = Stream.of(
	new Customer("Fred", "North")
	)
.collect(Collectors.groupingBy(Customer::getRegion));
```

Answer: `Map<String, List<Customer>>`

#### Partitioning
Another collector type allows "partitioning"
- this is effectively grouping by a `true/false` key

Example code:
```java
// Partition by, key is boolean  
List.of("Fred", "Jim", "Sheila", "Bill", "Joe", "Sally", "John", "Tommy")  
        .stream()  
        .collect(Collectors.partitioningBy(s -> s.length() > 4))  
        .forEach((k, v) -> {  
          System.out.printf("%s=%s\n", k ,v);  
        });
```
