#### Instance Methods
- Do not have the `static` modifier
- The first (usually implicit) argument is
	- called `this`
	- is of the enclosing type
	- is effectively `final`
- Must be invoked with an `instance` of the enclosing type as a prefix. This might be
	- implicitly the `this` value of the invoking instance method
	- explicitly the `this` value of the invoking instance method
	- any non-null expression referring to an instance of the enclosing type

#### Instance Method Invocation
Instance method invocation exhibits *Dynamic method invocation, virtual binding, or late binding* 
- If overridden method exist, the *type of the invoking object,* and *not the type of the reference,* determines the method to execute at *runtime*, not at compile time.

##### What happens?
```Java
void doStuff() {
	this = new Thing();
}
```
Answer: Compilation fails.
`this` is effectively `final`

 How can doStuff be invoked at line n1?
```Java
class Thing {
	void doStuff() {
	}
}
class Thing2 {
	void moreStuff() {
		// line n1
	}
}
```
Answer: An instance of `Thing` is needed
```Java
new Thing().doStuff();
```
or
```Java
Thing t = new Thing();
t.doStuff();
```

#### The Receiver Parameter
Instance methods typically have an implicit parameter `this` which is a reference to the invoking object.
- To permit annotation, `this` can be declared explicitly
Declare in a class called `Receiver`, these are equivalent:
```Java
void doStuff(String x) {}
void doStuff(Receiver this, String x) {}
void doStuff(@NotNull Receiver this, String x) {}
```
- The `this` value is effectively final, but the final modifier cannot be used

- Must be the first listed parameter
- Must be of the immediately enclosing type
- Must be called `this`
- Although `this` is effectively final, it's common for instance methods to mutate the object to which `this` refers

#### Arguments are Passed by Value
Java method arguments are always passed by value
- But the value of a reference type expression is a *reference*

##### What happens?
```Java
static void adder(int x) {
	x += 1;
}
public static void main(String[] args) {
	int x = 1;
	adder(x);
	System.out.println(x);
}
```
Answer: prints 1
