#### Sorting Arrays with `Arrays.sort`
Sort arrays using overloaded `Arrays.sort` methods 
- for all primitives, and sub-ranges 
- for object types, and sub-ranges, by natural order 
- for object types, and sub-ranges, by supplied `Comparator` order

#### Sorting Lists
Several mechanisms are provided for sorting lists
- List.sort(comparator)
- Collections.sort(list)
- Collections.sort(list, comparator)
Collections provides some related behaviors
- reverse (list)
- rotate(list, distance)
- shuffle(list), shuffle(list, random)
- swap(list, index1, index2)

##### Quick Question
Name three (including overloads) methods, and their argument
types, that can be used to put a list of strings into alphabetical
(unicode) order?

Answer:
- `List.sort(Comparator c)`
- `Collections.sort(List 1)`
- `Collections.sort(List 1, Comparator c)`
